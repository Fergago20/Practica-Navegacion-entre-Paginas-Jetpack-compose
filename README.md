📱 Práctica de Navegación - Jetpack Compose
Este proyecto es una aplicación de Android desarrollada con Kotlin y Jetpack Compose que demuestra cómo implementar la navegación entre diferentes pantallas (Destinations) de manera eficiente.

🚀 Características
Navigation Component: Implementación de NavHost, NavController y NavGraph.

Paso de Argumentos: Transferencia de datos entre pantallas de forma segura.

Arquitectura Moderna: Uso de funciones Composable para definir la interfaz de usuario.

UI Declarativa: Diseño moderno y fluido utilizando los estándares de Material Design 3.

🛠️ Tecnologías Utilizadas
Lenguaje: Kotlin

Framework UI: Jetpack Compose

Navegación: androidx.navigation:navigation-compose

Herramienta de Construcción: Gradle (Kotlin DSL / Groovy)
